# Tópicos Importantes

- Qual a linguagem de programação utilizada na disciplina?  
    Javascript

- O que é Node? 
    É o que permite rodar o Javascript em um servidor

- O que é Framework? e Qual a diferença para uma biblioteca?
    Framework é um conjunto de recursos para resolver um problema, a biblioteca pode ser utilizada de forma mais livre sem restrições

- O que é npm? 
    Gerenciador de pacotes do Node

- O que é React? 
    Uma biblioteca de código aberto de Javascript

- O que é o React Native?
    Biblioteca de código aberto para o desenvolvimento de aplicativos móveis

- O que é o Expo?
    Utiliza em conjunto com o React Native, simplifica a criação de aplicativos móveis
    
- Como estilizar um componente no React Native?
    Usando um objeto de estilos criado com StyleSheet (mais recomendado e performático) ou aplicando estilos diretamente no componente (inline).


- O estilo do React Native é igual ao CSS?
    Não, mas é inspirado nele, tendo sintaxes, unidades, heranças, posicionamentos e propriedades diferentes
